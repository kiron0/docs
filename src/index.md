## Quick Example

```javascript
run greet(name) {
  return `Hello, ${name}! Welcome to NullScript! üé≠`;
}

fixed message = greet("Developer");
speak.say(message);
```

Compiles to clean JavaScript:

```javascript
function greet(name) {
  return `Hello, ${name}! Welcome to NullScript! üé≠`;
}

const message = greet("Developer");
console.log(message);
```
